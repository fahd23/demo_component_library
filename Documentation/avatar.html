Avatar

Avatars are user profile pictures, which is usually found in circular shapes. You can also find squared avatar here.

Basic Avatar

Add classname "avatar" to any img element to make it an avatar.

Sized Avatar

Add classname "avatar" to the img element with "sm" or "lg" or "md" to make different sized avatar.

Squared Avatar

Add classname "avatar" to the img element with "sq-avtr" to make squared avatar. To make in different sizes, add "sm" or "lg" or "md".
